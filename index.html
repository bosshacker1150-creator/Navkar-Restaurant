User Panels

  • Browse menu, customize quantities, and place table-linked orders.
  • Track status: received → preparing → ready → served.
  • Pay online or at counter and submit ratings.
  • Accept/reject incoming orders and update status in real time.
  • Handle table requests and service notifications.
  • Coordinate with kitchen and mark served orders.
  • Manage menu, categories, staff, tables, and availability.
  • Monitor live operations: active tables, order flow, and revenue.
  • Generate reports: daily/weekly sales, top dishes, table activity.

Firebase Data Model

users/{userId}: { name, role }
menu/{itemId}: { name, price, category, image, available }
orders/{orderId}: { tableNumber, items, total, status, timestamp }
tables/{tableId}: { status }

Recommended security: only admins edit menu, waiters update order status, customers create orders.
